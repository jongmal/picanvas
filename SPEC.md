## 2. 조립 구조

### 모델 A - 회전 액자 어셈블리
```
[전면도]
┌─────────────────────────────────┐
│  ┌───────────────────────────┐  │
│  │                           │  │  3mm 베젤 오버랩
│  │      15.6" IPS 패널       │  │  (전면 립)
│  │      350.66 x 216.25      │  │
│  │                           │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘

[후면도]
┌─────────────────────────────────┐
│         │           │           │
│         │  ┌─────┐  │           │
│─────────┤  │중앙 │  ├───────────│  십자 보강재
│         │  │브라켓│  │           │
│         │  └─────┘  │           │
│         │           │           │
└─────────────────────────────────┘
```

#### 프레임
- 측벽: LCD 측면 감싸기 (3.2mm 두께) + 전면 립 (3mm 오버랩)
- 벽 두께: ~5-6mm
- 후면 클립으로 LCD 고정

#### 십자 보강재 (후면)
- 수평 + 수직 바가 프레임 후면을 가로지름
- 중앙: 정사각형 브라켓 플레이트

#### 중앙 브라켓
- 턴테이블 베어링 회전판에 체결 (4x M4)
- GT2 80T 풀리 링 일체형 (외치)
- 풀리 평면은 턴테이블 평면보다 전면 쪽으로 오프셋 (Z축 적층)

#### 후면 탑재 부품 (프레임과 함께 회전)
- Raspberry Pi 4B
- eDP 컨트롤러 보드
- TMC2209 드라이버 (방열판 포함)
- 5V 스텝다운 모듈
- 25V 470uF 전해 콘덴서
- 유리관 퓨즈 홀더

### 모델 B - 벽면 마운트 브라켓
```
[측면도 - 벽면에서 전면 방향]

벽면 ── 모델B ── 턴테이블(고정) ── 턴테이블(회전) ── 모델A
          │           10.5mm                              │
          │                                               │
          └── 모터 마운트 (17HS4023)       80T 풀리 링 ──┘
               └── 20T 풀리 ─── GT2 벨트 300mm ──┘
```

#### 벽면 플레이트
- 콘크리트 칼블록에 앵커 고정
- 턴테이블 베어링 고정판 체결 (4x M4)

#### 모터 마운트
- 측면 클램프 방식 (모터 축 수직, 샤프트 위 방향)
- 20T 풀리가 모델 A의 80T 풀리와 동일 Z 높이에 위치
- 축간 거리: 프레임 중심에서 ~98mm

## 3. 배선도

### 전원 분배
```
[65W USB-C PD 충전기]
        │
        │ USB-C
        ▼
┌──────────────┐
│  PD 디코이   │──── 12V 출력 (납땜)
│  (12V)       │
└──────┬───────┘
       │
       ├─── A: 5V 스텝다운 ──► Raspberry Pi 4B (USB-C 5V)
       │
       ├─── B: 12V 직접 ────► 스테퍼 모터 (TMC2209 경유)
       │                       └── 25V 470uF 캡 (전원 평활)
       │                       └── 유리관 퓨즈 (피크 1.5배)
       │
       └─── C: 12V 직접 ────► eDP 컨트롤러 보드 (USB-C 12V)
```

### 신호선
```
Raspberry Pi 4B
    ├── GPIO ──► TMC2209 (STEP/DIR 또는 UART) ──► 17HS4023 모터
    │
    └── micro HDMI ──► HDMI 케이블 ──► eDP 컨트롤러 ──► eDP 40pin ──► 15.6" IPS
```

### 케이블 진입
- USB-C 케이블 1개가 **프레임 하단**으로 진입
- 내부 여유 루프로 90° 회전 대응

## 4. 회전 메커니즘

| 항목 | 값 |
|------|-----|
| 회전 범위 | 90° (가로 ↔ 세로) |
| 구동 | 17HS4023 → 20T 풀리 → GT2 벨트 → 80T 풀리 |
| 감속비 | 4:1 |
| 베어링 | 57mm 정사각형 턴테이블 (하중 지지) |
| 프레임 측 유효 토크 | ~48 N·cm |

## 5. 3D 프린팅 제약

| 항목 | 값 |
|------|-----|
| 프린터 | Bambu A1 Mini |
| 최대 출력 크기 | 180 x 180 mm |
| 소재 | 미정 (PLA/PETG) |
| 분할 방식 | 180mm 초과 파트는 도브테일 조인트로 분할 |

### 모델 A 분할 계획
- 프레임: 350.66 x 216.25mm → 양축 모두 180mm 초과
- 십자 바: 중앙 브라켓 접합부에서 분할
- 최소 4개 프레임 세그먼트 + 중앙 브라켓 + 십자 세그먼트

### 모델 B
- 180mm 이내 가능성 높음 (턴테이블 57mm + 모터 42mm + 간격)
- 단일 파트 또는 2분할

## 6. Z축 적층 순서 (벽면 → 전면)

```
Layer 0: 벽면
Layer 1: 모델 B 벽면 플레이트 (~5mm)
Layer 2: 턴테이블 베어링 고정판 (10.5mm의 일부)
Layer 3: 턴테이블 베어링 회전판 (10.5mm의 일부)
Layer 4: 모델 A 중앙 브라켓 + 80T 풀리 링 (~8mm)
Layer 5: 십자 보강재
Layer 6: 부품 장착 영역 (Pi, 보드류)
Layer 7: LCD 패널 (3.2mm)
Layer 8: 프레임 전면 립 (3mm 베젤)
```

총 벽면~전면 깊이: ~45-55mm (추정)
