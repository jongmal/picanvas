# PiCanvas - 프로젝트 컨텍스트

## 프로젝트 개요
15.6인치 IPS 디스플레이를 사용하는 **벽걸이 회전형 디지털 액자**.
Raspberry Pi 4B 기반으로 가로/세로 90° 자동 회전, API 디지털액자 기능, SD카드 메타데이터 표시 예정.

## 핵심 설계 결정

### 1. Z축 적층 구조 (벽면 → 전면)
```
벽면 → 모델B(5mm) → 턴테이블 고정판(10.5mm) → 회전판 → 모델A 중앙브라켓(5mm) + 80T풀리(8mm) → 십자 보강 → 부품 → LCD(3.2mm) → 베젤 립(3mm)
```
- 턴테이블과 풀리가 같은 평면에서 겹치는 문제 → **Z축 오프셋으로 해결**
- 턴테이블이 하중 담당, 풀리가 회전력 전달

### 2. 프레임 분할
- 원래 4분할 → 가로 181.33mm로 180mm 초과
- **6분할(3x2)로 변경** → 각 세그먼트 120.89 x 114.13mm
- 각 분할선에 도브테일 조인트

### 3. 80T 풀리 일체형
- GT2 80T 외치 풀리가 중앙 브라켓에 일체 성형
- 축 없이 턴테이블 회전판에 직접 체결
- 피치 직경 ~50.93mm, 벨트 폭 6mm + 상하 가이드 플랜지

### 4. 벨트 300mm 폐루프
- 축간 거리 ~98mm (20T↔80T)
- 감속비 4:1, 유효 토크 ~48 N·cm

### 5. 모터 배치
- 17HS4023 팬케이크 모터, 축 수직 (샤프트 위 방향)
- 모델 B에서 측면 클램프로 고정
- 높이 최소화 목적

| belt_center_distance | 98 mm | 풀리 축간 거리 |

## 배선 요약
```
65W USB-C → PD 디코이(12V) → A: 5V스텝다운→Pi / B: 12V→TMC2209→모터 / C: 12V→eDP보드
Pi GPIO → TMC2209(STEP/DIR) → 모터
Pi micro HDMI → eDP 컨트롤러 → eDP 40pin → 15.6" IPS
```
